The following keyboard interactions are available:

<KeyboardControls>
<KeyboardControl keyOrCombos="Tab">

If focus is above the list, moves focus to one of the following items depending on previous user interaction:

- The last selected item
- The last item with focus
- The first item in the list

</KeyboardControl>
<KeyboardControl keyOrCombos="Shift + Tab">

If focus is below the list, moves focus to the list item that last had focus or the first item in the list.

</KeyboardControl>
<KeyboardControl keyOrCombos="Space / Enter">

When focus is on a list item, selects the item and deselects the previously selected item, if any.

</KeyboardControl>
<KeyboardControl keyOrCombos="ArrowUp">

Moves focus to the previous item in the list. If focus is on the first item, the next ArrowUp key press retains focus on the first item. It doesn't wrap focus to the last item in the list.

</KeyboardControl>
<KeyboardControl keyOrCombos="ArrowDown">

Moves focus to the next item in the list. If focus is on the last item, the next ArrowDown key press retains focus on the last item. It does not wrap focus back to the first item in the list.

</KeyboardControl>
<KeyboardControl keyOrCombos="Home">

When focus is within the list, moves focus to the first list item.

</KeyboardControl>
<KeyboardControl keyOrCombos="End">

When focus is within the list, moves focus to the last list item.

</KeyboardControl>
</KeyboardControls>
